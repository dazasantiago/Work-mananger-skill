# Project Management

Módulo encargado del CRUD ad-hoc de tasks/proyectos y la revisión rápida de
pendientes, fuera del flujo de sesión.

## Router

| Acción del usuario | Archivo |
|---------------------|---------|
| Agregar, editar, completar o eliminar tareas | `modules/project/task-crud.md` |
| Agregar o actualizar proyectos, cambiar estado, capturar idea | `modules/project/project-crud.md` |
| Ver qué hay pendiente sin empezar sesión | `modules/project/quick-review.md` |

Lee únicamente el archivo correspondiente a la acción — cada uno es autosuficiente.
